How to run the code:
step 1: Enter (clear) command.
step 2: Load the "facts-trial.clp" file.
step 3: Load the "rules-trial.clp" file.
step 4: Enter (reset) command.
step 5: Enter (run) command.
step 6: Check for the text file s-table.txt in the same folder after each run.

After running with each data set, you will find a text file s-table containing the simulation results from each run one below the other.

The project has been built and compiled using CLIPS 6.3 on OS Windows 8.
